# START HERE｜口碑雷达快速入口

## 1. 30 秒开箱即用

### 简版模式

复制这句话即可开始：

```
请用简版模式分析以下评论，只输出 Top 3 问题、高风险提示和今日三项行动：
[在这里粘贴 5–10 条评论]
```

### 完整版模式

复制这句话即可开始：

```
请用完整版模式分析以下评论和客服记录，输出完整 9 个模块，并保留证据编号、风险评分、置信度和今日三项行动：
[在这里粘贴评论 / 客服记录 / 售后反馈]
```

### 只生成差评回复

```
请只根据以下差评生成真诚、克制、合规的回复话术，并说明哪些情况需要人工介入：
[在这里粘贴差评]
```

## 2. 最小完整案例

### 输入

R01【负向】等了五天才发货，客服只说帮忙催一下。
R02【负向】颜色比图片深很多，有点失望。
R03【中性】包装有点破，但商品本身还可以。
R04【负向】客服态度不错，但没有解决问题。
R05【正向】版型好看，价格也能接受。

### 输出摘要

* Top 1：发货慢 / 客服无闭环，风险等级高，置信度中；
* Top 2：颜色预期不一致，风险等级中，置信度中；
* Top 3：包装破损，风险等级中，置信度低；
* 今日行动：

  * P0：人工跟进发货异常用户，并给出明确回访时间；
  * P1：修改详情页颜色说明，补充自然光实拍提示；
  * P2：新增包装破损处理 FAQ。

说明：该案例只有 5 条评论，适合第一次体验简版模式。由于样本量较小，风险判断应标注中 / 低置信度，不应输出复杂经营结论。

## 3. FAQ 摘要

| 问题                        | 回答                                      |
| ------------------------- | --------------------------------------- |
| 只有几条评论能用吗？                | 可以，但少于 3 条时只输出低置信度简版判断，不输出复杂风险分。        |
| 一定要上传 Excel 吗？            | 不需要，直接粘贴评论即可。                           |
| 风险分是精确指标吗？                | 不是。风险分是基于文本证据的辅助估算，需要同时看置信度和原文证据。       |
| 能分析淘宝、抖店、小红书、Shopify 评论吗？ | 可以，但必须是用户主动粘贴或商家后台导出的文本，不自动爬取平台数据。      |
| 能帮我删差评、刷好评或控评吗？           | 不能。Skill 会拒绝违规请求，并提供合规回复和体验改进建议。        |
| 输入很乱怎么办？                  | Skill 会先做输入质量检查；无法稳定判断时会输出兜底响应，并提示补充数据。 |

## 6. 异常输入兜底

当输入为空、少于 3 条、乱码严重、跨商品混杂、无真实数据或完全无关时，Skill 不强行生成完整报告，而是输出：

1. 当前无法完整分析的原因；
2. 当前可以初步判断的内容；
3. 需要补充的数据；
4. 可复制输入模板；
5. 合规替代方案。

---

## 7. 可信边界

* 不接非授权平台 API；
* 不做网页爬虫；
* 不绕过登录或平台权限；
* 不生成刷评、控评、删评、诱导好评建议；
* 不编造退货率、投诉率、转化率或平台规则；
* 所有运营建议最终由商家人工确认。
