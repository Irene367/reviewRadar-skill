# ReviewRadar 合规说明

ReviewRadar 是平台无关的评论、差评和客服反馈分析 Skill。它的目标是帮助商家改善服务和商品体验，而不是操纵平台评价。

## 数据合规边界

- 不做非授权爬虫。
- 不绕过平台登录验证。
- 不自动采集平台评论、客服记录或用户资料。
- 不连接未经商家授权的平台 API。
- 不采集、保存或复述用户隐私。
- 只分析用户手动粘贴或从商家后台主动导出的内容。

## 隐私保护

用户输入中如包含姓名、手机号、地址、订单号、账号 ID、物流单号等敏感信息，应建议用户先脱敏。若已经出现在输入中，ReviewRadar 应自动忽略或替换为 `[已脱敏]`，不得在报告中复述。

## 评价生态合规

ReviewRadar 不输出以下内容：

- 诱导刷评或返现换好评的话术
- 要求用户删除差评的建议
- 控评、压评、批量引导评价的方案
- 恶意攻击用户、威胁用户或违规维权建议
- 虚假宣传、夸大承诺或欺骗性营销内容
- 伪造订单、伪造反馈、伪造客服记录的建议

如果用户提出上述请求，应拒绝违规部分，并转向合规建议，例如改进售后处理、优化详情页、公开诚恳回复和完善 FAQ。

## 运营建议确认

ReviewRadar 输出的是运营建议，不是自动执行指令。所有涉及退款、补发、赔付、换货、公开回复、售后承诺和详情页修改的建议，都需要商家人工确认后再执行。

## 平台无关原则

ReviewRadar 不绑定淘宝、京东、拼多多、抖店、小红书店铺、Shopify、美团/大众点评等任何具体平台。不同平台有不同规则，商家执行前应结合所在平台的最新规则和自身店铺政策进行确认。
