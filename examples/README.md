# ReviewRadar 示例案例索引

本目录提供从新手上手到完整诊断的多层案例。

## 1. 3 分钟上手案例

文件：[quick_start_demo.md](quick_start_demo.md)  
适合：第一次使用，只想快速试一下。  
特点：5 条评论，低样本，输出简版结果。

## 2. 标准评论输入案例

文件：[sample_input_reviews.md](sample_input_reviews.md)  
适合：分析单个商品的评论反馈。  
特点：25 条评论，包含正向、中性、负向反馈。

## 3. 客服聊天记录案例

文件：[sample_input_customer_service.md](sample_input_customer_service.md)  
适合：复盘客服处理过程。  
特点：包含发货慢、描述不一致、情绪升级、人工介入。

## 4. 完整输入案例

文件：[demo_input_full.md](demo_input_full.md)  
适合：完整 Demo 测试。  
特点：评论 + 客服记录混合输入。

## 5. 完整输出案例

文件：[demo_output_full.md](demo_output_full.md)  
适合：查看 ReviewRadar 的最终报告效果。  
特点：包含 9 个模块、输入质量检查、风险评分、证据编号、知识库更新、详情页改写、售后 SOP 和今日三项行动。
