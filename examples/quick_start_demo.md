# 3 分钟上手 Demo

## 使用方式

复制下面这句话和 5 条评论，粘贴给 ReviewRadar：

```text
请用简版模式分析以下评论，只输出 Top 3 问题、高风险提示和今日三项行动：

R01【负向】颜色比图片暗很多，详情页看着更浅。
R02【负向】平时 M 码，这件腰和胸都有点紧。
R03【负向】页面写 48 小时发货，结果第四天才发。
R04【中性】衣服版型还行，但面料没有想象中厚。
R05【正向】上身显瘦，长度对小个子挺友好。
```

## 预期输出

ReviewRadar 应输出简版结果：

1. 本次口碑总览；
2. Top 3 用户问题聚类；
3. 高风险差评预警；
4. 今日三项优先行动。

## 注意

该案例只有 5 条评论，适合快速体验。结论应标注低样本或低置信度，不适合直接作为重大经营决策依据。
